Requirements & Compatibility

Requirements

To run WordPress your host just needs a couple of things:

  • PHP version 5.5 or greater (recommended: PHP 5.6 or greater)
  • MySQL version 5.56 or greater (recommended: MySQL 5.6 or greater)

Having your server configured properly is crucial for the theme to function. Make sure that the following PHP server properties are set according to described below or contact your Host provider to support you.

  • max_execution_time 300
  • max_input_time 300
  • memory_limit 64M
  • upload_max_filesize 64M
  • max_input_vars 3000

NOTE: You can use plugin WordPress phpinfo() to check phpinfo from your server. Enough server configuration will also ensure fast speed and increase performance.

Common Theme Installation Problems

Issue 1: Get the message "Are you sure you want to do this?"

If you are installing your theme from the WordPress theme installer, but get a message that says Are you sure you want to do this? ,most likely your web server is configured with low PHP settings that only allow a certain size ZIP file to be uploaded via WP admin.

Most hosting companies configure their servers to only have the maximum file upload size limit at 8-10 MB, some are less. So if you get this error, your web server is rejecting the upload due to the overall size of the sweetinz.zip file. This is not a theme issue or bug.

» Solution: There two ways to solve this problem.

  1. Contact your hosting provider to increase your PHP limits so a larger zip file is allowed to be uploaded. They should increase your maximum upload size. You can read section Requirements & Compatibility again.
  2. You can upload your theme via FTP, see section Theme Installation via FTP. It is very easy and efficient.

Defining a Menu

We will see how to create your menu, you must define a menu before you can add items to it.

You follow the steps to create a new menu:

  1. Step 1:Login to the WordPress Dashboard.
  2. Step 2:From the Appearance menu on the left-hand side of the Dashboard, select the Menus option to bring up the Menu Editor.

  3. Step 3:Select the link Create a new menu at the top of the page.
  4. Step 4:Enter a name for your new menu in the Menu Name box.
  5. Step 5:Click the Create Menu button.

Now your new custom menu has now been defined and you can now add pages to it (steps below).

Adding Items to a Menu

You can add different link types into your menu, these are split between panes left of the menu you're currently editing.

  1. Step 1:Locate the pane entitled Pages.
  2. Step 2:Within this pane, select the View All link to bring up a list of all the currently published Pages on your site.
  3. Step 3:Select the Pages that you want to add by clicking the checkbox next to each Page's title.
  4. Step 4:Click the Add to Menu button located at the bottom of this pane to add your selection(s) to the menu that you created in the previous step.
  5. Step 5:Click the Save Menu button once you've added all the menu items you want.

Your custom menu has now been saved..

Assigning a Menu to a Location

Once you've created a menu and filled it with items, it's time to assign it to one of the theme pre-defined locations.

  1. Step 1:Scroll to the bottom of the menu editor window.
  2. Step 2:In the section titled Theme locations, click the check box for the location where you want your menu to appear.
  3. Step 3:Click Save Menu once you've made your selection.

Post Settings

WordPress allows you to switch on different post formats from a set list. You don't have to use all of them, you simply have to pick which formats suit your site best. on Sweetinz theme supports 7 post formats, this list consists of:

  • Aside:Similar to a quick note. Use these for posts like quick updates or custom status.
  • Gallery:A post containing a gallery of images.
  • Image:A single image, rather than a gallery.
  • Video:A post that has one, or several, videos. The WordPress codex also points out that a single URL may indicate the source of a video, rather than an embedded video.
  • Audio:An audio file or playlist.
  • Link:A simple link to an external site.
  • Quote:A quoted block of text, usually with attribution to a quote's author.

Translations

Sweetinz theme is ready to translate. It is possible to translate every single string that the theme outputs on the frontend to any language you like. Also Sweetinz was tested with WPML plugin and is fully compatible. If you want to have multilingual website, this is the best way to do it. Also if you want free plugin you can use Polylang plugin.

Sweetinz allows you to translate the theme into any language. There are 2 methods to do:

  1. Use the Poedit software and follow this document.
  2. Use the translation plugins, such as Loco Translate plugin.

Translation via .PO file

  1. Step 1:Download and install POEdit translation editor (https://poedit.net/)
  2. Step 2:Go to the theme translation folder (sweetinz/languages/)
  3. Step 3:Make a copy of the file "sweetinz.pot" and rename the file to the locale of your language. For example, Italian would be it_IT.po. See language code list here WordPress in Your Language
  4. Step 4:Open POEdit and load the file. (i.e it_IT.po)
  5. Step 5:Translate all the strings in the list and save changes. After you save, you will have two files. (i.e. it_IT.po and it_IT.mo)
  6. Step 6:Upload these files to sweetinz/languages/
  7. Step 7:For WordPress 4.0 and above, go to Settings » General and select Site Language
  8. For WordPress 3.9.2 and below, open your wp_config.php file in the root, and find this line: define('WPLANG', ''); Change it into your language code. For example, define('WPLANG', 'it_IT');

Translation via LOCO Plugin

We will using Loco Translate plugin that provides in-browser editing of WordPress translation files. To translate the theme to your language, please follow these steps:

  1. Step 1:Install and activate LOCO Translate plugin.
  2. Step 2:Create and update language files directly in your theme or plugin.
  3. Step 3:Saving custom your language translations.

Please consult our video guide on How to translate Catanis Theme using LOCO Translate plugin, We have been use Luvaniz theme for example and the process of Sweetinz theme translation is similar to this.

WPML Integration

Sweetinz is WPML ready! This means that you can translate pages, posts, custom post types, taxonomies, menus and more. If you want to use this plugin, you must purchase it first and then install into your wordpress installation.

If you are new to WPML, we strongly suggest you browse through the WPML documentation. Below is a list of links that we feel would be a good starting point.

How To?

1. How to enable coming soon page?
Please go to Theme Options » General Settings » Coming Soon Page, and change 2 fields:
  1. Field Enable Under Construction: Turn ON
  2. Field Under Construction Page: Choose a page for it

After you can change some options below such as coming soon style, logo, background..
2. How To change color scheme in each page?
Please go to and edit that page, scroll down you will see CATANIS PAGE SETTINGS » Layout/Header tab » Color Scheme and choose a color you want.

5. How to Override CSS?
Have 3 ways to override css for theme:

1. Use Customize in WordPress: Please go to Appearance » Customize » Additional CSS

2. Use Theme Options: Please go to Theme Options » General Settings » Custom Code » Custom CSS

3. Use child theme: You add your custom code in style.css file in child theme

6. How to add Goolge/Twitter API Keys?

Please go to Theme Options » General Settings » API Keys
